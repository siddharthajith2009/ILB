STEP 1: Open Terminal in Visual Studio Code
--------------------------------------------
- In VS Code, go to: Terminal ‚Üí New Terminal
- Or press: Ctrl + ` (backtick key)


STEP 2: Install Required Packages
----------------------------------
Copy and paste this command into your terminal:

    python3 -m pip install pandas numpy scikit-learn

Or if you prefer using the requirements file:

    python3 -m pip install -r requirements.txt


STEP 3: Run the Preprocessing Script
-------------------------------------
Navigate to the src folder and run the script:

    cd src
    python3 preprocess_data.py


EXPECTED OUTPUT:
----------------
You should see:
‚úì Dataset loaded message
‚úì Missing values report
‚úì Outlier detection results
‚úì Encoding progress for each categorical feature
‚úì Normalization progress for each numerical feature
‚úì Summary report
‚úì Success message

The script will create a new file:
üìÅ data/preprocessed_job_data.csv


TROUBLESHOOTING:
----------------
‚ùå "No module named 'pandas'"
   ‚Üí Make sure you ran the pip install command first
   ‚Üí Use: python3 -m pip install pandas numpy scikit-learn

‚ùå "File not found" error
   ‚Üí Make sure you're in the correct directory
   ‚Üí Check that the CSV file is in the data/ folder

‚ùå Permission denied error
   ‚Üí Try adding --user flag: python3 -m pip install --user pandas numpy scikit-learn


VERIFYING INSTALLATION:
-----------------------
To check if packages are installed correctly:

    python3 -c "import pandas; import numpy; import sklearn; print('‚úì All packages installed!')"
